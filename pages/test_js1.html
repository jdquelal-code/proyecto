Prueba JavaScript Externa

¿Qué hace este código?

Este archivo HTML utiliza la etiqueta <script src="..."> para enlazar un archivo de lógica independiente ubicado en la carpeta js/app.js. El atributo defer asegura que el script se ejecute solo después de que el HTML se haya cargado por completo.

Ejemplo del archivo app.js:

// Contenido de ../js/app.js
console.log("¡Hola desde el archivo externo!");
alert("JavaScript cargado correctamente ✅");
